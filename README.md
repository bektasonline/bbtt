# BBTT - CloudStream Custom Extensions Repo

Bu depo, CloudStream app'i için özel eklentiler (extensions) içerir.  
Özellikle Türkçe siteler (örneğin Dizibox) için scraper'lar.

## Kurulum
1. CloudStream'de Ayarlar → Extensions → Repositories → Add Repository.
2. URL: `https://github.com/bektasonline/bbtt/raw/main/repo.json`

## Eklentiler
- **Dizibox**: Dizi/film arama ve stream.

## Geliştirme
- Kotlin ile eklenti yaz.
- Build: `./gradlew make`
